**LANL Earthquake Prediction (Can you predict upcoming laboratory earthquakes?)** Author: [__Abhishek Ranjan__](https://ca.linkedin.com/in/aranjanlinkedin), May 2019 This repo contains source code and documentation for my Silver medal winning solution ([__214th place - top 5%__](https://www.kaggle.com/c/LANL-Earthquake-Prediction/leaderboard)) for [__LANL Kaggle competition__](https://www.kaggle.com/c/LANL-Earthquake-Prediction/overview). The algorithm uses pyramidal, multi-scale, LBP features (see my other prize winning solution for [__details of this feature__](https://github.com/aranjandev/vsbsigproc/blob/master/docs/VSB-Power-Line-Fault-Detection.pdf)) with gradient boosting algorithm [__XGBoost__](https://xgboost.readthedocs.io/en/latest/).
